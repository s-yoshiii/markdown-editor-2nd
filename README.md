## markdown-editor
React & Typescipt with chakraUIで構築したシンプルなマークダウンエディタです。

## 機能
- テキスト入力
- Markdownリアルタイムプレビュー
- テキストの保存・復元（IndexDB）

## 目的
React & Typescript学習用
